Para as portas NOT, NAND E NOR, torna-se necessaria uma decodificacao de 
valores.

output   correct
  -1        1
  -2        0
  
Devido 'a necessidade de novas instrucoes, duas CPU foram criadas.
Os arquivos "Wolbart_new_1.html" e "Wolbart_new_1.cpu" correspondem
'as questoes que envolvem portas logicas (NOT, XOR, NAND, AND, NOR).
  
Para as questoes de Fibonacci, Numeros primos e M.D.C., utilizar os arquivos 
"Wolbart_new_2.html" e "Wolbart_new_2.cpu".
  
Para a questao de M.D.C., digitar somente numeros positivos.
